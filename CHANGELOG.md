## [v0.15.0] - 2026-05-03

### Highlights

- PDF background image with text overlay, multi-page support, and per-segment styling
- Security hardening: secure file operations (G703), secure cookie attributes (G124)

### Added

- `pdf`: add `CreateBackgroundPDF()` and `CreateBackgroundPDFFile()` for PDFs with background image, logo, and text overlay ([`e20a22a`](https://github.com/grokify/sogo/commit/e20a22a))
- `pdf`: add `CreateMultiPagePDF()` and `CreateMultiPagePDFFile()` for multi-page PDF generation ([`66f3fb5`](https://github.com/grokify/sogo/commit/66f3fb5))
- `pdf`: add `TextBlock` type for per-segment style overrides (font, size, color, alignment) ([`66f3fb5`](https://github.com/grokify/sogo/commit/66f3fb5))
- `pdf`: add `TextAlign` type with `Left`, `Center`, `Right` alignment options ([`66f3fb5`](https://github.com/grokify/sogo/commit/66f3fb5))
- `pdf`: add `Page` struct for per-page configuration in multi-page PDFs ([`66f3fb5`](https://github.com/grokify/sogo/commit/66f3fb5))
- `pdf`: add logo overlay support with configurable position and scaling ([`e20a22a`](https://github.com/grokify/sogo/commit/e20a22a))
- `pdf`: add markdown parsing helpers for text segments ([`e20a22a`](https://github.com/grokify/sogo/commit/e20a22a))

### Fixed

- `pdf`: avoid deferred file handle accumulation in `CreateMultiPagePDFFile` loop ([`59fec6b`](https://github.com/grokify/sogo/commit/59fec6b))
- `markdown`: use `osutil.ReadFileSecure`/`WriteFileSecure` to prevent path traversal (G703) ([`30170ae`](https://github.com/grokify/sogo/commit/30170ae))
- `anyhttp`: add `Secure`, `HttpOnly`, `SameSite=Strict` to `http.Cookie` (G124) ([`9bc313f`](https://github.com/grokify/sogo/commit/9bc313f))
- `cobrautil`: use `reflect.Pointer` instead of deprecated `reflect.Ptr` ([`141d123`](https://github.com/grokify/sogo/commit/141d123))

### Tests

- `pdf`: add comprehensive tests for background PDF generation, logo overlay, markdown parsing ([`d4337fa`](https://github.com/grokify/sogo/commit/d4337fa))
